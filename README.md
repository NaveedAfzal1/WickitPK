## What It Does
WickitPK eliminates PSL ticket fraud and scalping using blockchain:
- Unfakeable NFT tickets on WireFluid
- Smart contract price caps prevent scalping
- Attended matches become permanent digital collectibles
- Fan loyalty badges (Bronze/Silver/Gold/Diamond)
- Gift tickets to family and friends
